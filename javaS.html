Que es JavaScript?

JavaScript es un lenguaje de programación que se utiliza para crear páginas web interactivas y
dinámicas, añadiendo características
como animaciones, formularios que responden a acciones del
usuario y actualizaciones de contenido sin recargar la página. Trabaja junto con HTML y CSS
para crear experiencias de usuario complejas y es la tecnología principal del lado del cliente en la World Wide Web,
ejecutándose directamente en el navegador del usuario.

Interactivo

Interactividad: Permite la creación de elementos interactivos como juegos, menús desplegables y carruseles de imágenes.
Actualizaciones en tiempo real: Puede actualizar partes de una página web sin necesidad de recargarla por completo, como sucede en las redes sociales con las notificaciones.
Validación de formularios: Verifica que la información que un usuario introduce en un formulario sea correcta antes de enviarla al servidor.
Animaciones: Permite animar elementos de la página, como hacer que se muevan, cambien de tamaño o aparezcan y desaparezcan.
Contenido dinámico: Se utiliza para mostrar contenido que cambia según la hora, la ubicación o la interacción del usuario.

¿Cómo funciona?

El código JavaScript se puede incrustar
directamente en una página HTML o en un archivo .js separado. Cuando el navegador descarga la página,
ejecuta el script de JavaScript junto con el código HTML y CSS.
Esto sucede en el dispositivo del usuario, a diferencia de los lenguajes del lado del servidor, que son
procesados por el servidor antes de enviar el resultado al navegador.

¿Es lo mismo que Java?

No, aunque los nombres son similares, Java y JavaScript son lenguajes de programación diferentes.
Java: Es un lenguaje compilado y orientado a objetos que suele usarse en aplicaciones del lado del servidor.
JavaScript: Es un lenguaje de scripting interpretado, más flexible y utilizado
principalmente para el desarrollo del lado del cliente (en el navegador).
