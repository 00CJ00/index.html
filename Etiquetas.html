Que son Etiquetas en HTML?

Las etiquetas en HTML son fragmentos de código que se utilizan para
crear elementos y estructurar el contenido de una página web. Indican al
navegador web cómo debe interpretar
y mostrar el texto y otros elementos (como imágenes, enlaces, listas, etc.). Propósito principal Estructurar contenido: Definen la
jerarquía y organización del contenido, como títulos,
párrafos, listas y secciones. Indicar visualización: Le dicen
al navegador cómo debe renderizar visualmente el contenido
(por ejemplo, poner texto en negrita, crear un enlace o mostrar una imagen). Estructura de una etiqueta La mayoría de las etiquetas vienen
en pares: una de apertura y una de cierre, que envuelven el
contenido al que afectan. Etiqueta de apertura: Comienza
con un símbolo de menor que (<), seguido brdel nombre de la etiqueta termina con un símbolo de mayor que (>). Etiqueta de cierre: Es similar a la de apertura,
pero incluye una barra inclinada (/) antes del
nombre de la etiqueta (). Contenido: Es la información que se coloca entre las etiquetas de apertura y cierre.

Ejemplo:

p> es la etiqueta de apertura.
/p> es la etiqueta de cierre.
"Este es un párrafo de texto." es el contenido.

Estructura y contenido

html>: La etiqueta raíz de un documento HTML.
head>: Contiene metadatos, títulos y enlaces a otros archivos.
body>: Contiene el contenido visible de la página.
main>: Engloba el contenido principal del documento.
footer>: Define el pie de página de un documento.
section>: Define una sección de contenido.
Texto y formato p>: Define un párrafo de texto.
h1> a h6>: Definen títulos de diferentes niveles.
strong>: Indica texto importante o en negrita.
em>: Enfatiza un texto.
br>: Inserta un salto de línea.
hr>: Crea una línea horizontal para separar contenido.
span>: Un contenedor genérico para texto en línea.
pre>: Muestra texto preformateado.
Enlaces a>: Crea hipervínculos a otras páginas o secciones.
img>: Muestra imágenes.
audio>: Inserta contenido de audio.
video>: Incrusta videos.
Listas ul>: Crea una lista no ordenada (de puntos).
ol>: Crea una lista ordenada (numerada).
li>: Define cada elemento de una lista.
Tablas
table>: Define una tabla.
tr>: Crea una fila en la tabla.
td>: Define una celda de datos en la tabla.
th>: Define una celda de encabezado en la tabla.
caption>: Define el título de la tabla.
Formularios
form>: Define un formulario.
input>: Crea elementos de entrada como campos de texto, botones, etc.
label>: Define una etiqueta para un elemento de formulario.
textarea>: Crea un área de texto de varias líneas.
button>: Define un botón.
Otros
div>: Un contenedor genérico para agrupar elementos y estructurar el contenido.
span>: Un contenedor para agrupar texto o elementos en línea.
style>: Define estilos CSS.
script>: Incluye código JavaScript.
iframe>: Inserta otro documento HTML dentro del documento actual.
<-- -->: Se utiliza para comentarios en el código.
meta>: Contiene metadatos sobre el documento.
