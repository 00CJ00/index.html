¿Qué es HTML?

HTML son las siglas de HyperText Markup Language (Lenguaje de Marcado de Hipertexto) y es el
lenguaje estándar para crear y estructurar el contenido de las páginas web. Utiliza etiquetas
para definir elementos como texto, imágenes, videos y enlaces, que un navegador web interpreta y
muestra para formar una página.
Es un lenguaje de marcado, no de programación, y es fundamental para
el desarrollo web, sirviendo como la
base para tecnologías como CSS (para estilos) y JavaScript (para interactividad).

Historia HTML

HTML (Lenguaje de Marcado de Hipertexto) fue creado por
Tim Berners-Lee en 1991 para permitir a los científicos
compartir documentos. Desde su primera versión en 1991 y su publicación oficial en 1993, ha evolucionado a través de varias
versiones, incluyendo HTML 4.01 y XHTML, hasta llegar a HTML5, que se ha convertido en el
estándar actual. HTML5 introdujo mejoras para la creación de sitios web
multimedia e interactivos, y para la compatibilidad con dispositivos móviles. Origen y primeras versiones 1980: Tim Berners-Lee propuso un sistema de hipertexto para compartir documentos. 1991: Berners-Lee publicó la primera versión oficial de HTML,
que se convirtió en la
base del proyecto World Wide Web (WWW). 1993: Se lanzó una versión de
HTML escrita por Tim Berners-Lee. 1995: HTML 2.0 estandarizó características que ya se estaban usando. Evolución y versiones posteriores HTML 4.01: Fue una versión muy utilizada y
se convirtió en estándar oficial en diciembre de 1999. XHTML: Fue una reformulación de HTML basada en XML,
con reglas más estrictas. No tuvo el éxito esperado debido a su rigidez, y el
desarrollo se canceló en 2006. HTML5: Lanzado en 2014, fue un punto de inflexión
que permitió la integración nativa de elementos multimedia como video> y audio>,
mejoró la semántica con etiquetas como article> y section>, y potenció la
compatibilidad con dispositivos móviles. Características clave en su evolución Hipertexto: Su característica fundamental desde el principio,
permitiendo enlazar documentos. Etiquetas: Las etiquetas son el
componente esencial que da estructura a los documentos. Nuevos elementos: Cada versión ha añadido nuevos elementos para enriquecer la web,
desde tablas e imágenes hasta video y audio. Compatibilidad móvil: Las versiones más recientes se enfocan en la adaptabilidad a
dispositivos móviles, utilizando técnicas como el diseño responsivo.

Editores de HTML

Los editores de HTML son herramientas de software para crear y editar código HTML, como Visual Studio Code, Sublime Text y Notepad++. Se clasifican en editores de texto, que permiten codificación directa, y editores WYSIWYG ("lo que ves es lo que obtienes"), que ofrecen una interfaz de diseño visual, como Adobe Dreamweaver. Editores de texto (para codificación directa)
Visual Studio Code: Gratuito, potente y con muchas extensiones.
Sublime Text: Es un editor ligero, rápido y popular.
Notepad++: Es una opción gratuita, ligera y eficiente, pero principalmente para Windows.
Atom: Gratuito y muy personalizable, con un diseño moderno.
Bluefish: Es un editor potente, ligero y gratuito, diseñado para desarrolladores web.

Editores WYSIWYG (diseño visual) Adobe Dreamweaver: Un programa comercial con un enfoque visual y herramientas integradas.
Brackets: Gratuito, de código abierto, enfocado en diseño web con vista previa en tiempo real.
